# SecureRelayFL

Privacy-preserving federated learning for adaptive relay protection settings over industrial communication networks.

## About

SecureRelayFL evaluates federated learning under realistic industrial network impairments for multi-task fault classification and protection relay setting optimization across heterogeneous facilities — without sharing proprietary operational data.
